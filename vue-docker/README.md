> Docker compose file

## 👍 Usage

```sh
// build containers
$ docker-compose build

// start containers(in background)
$ docker-compose up -d

// enter container's bash
$ docker-compose exec web /bin/sh

// just started
$ vue create {app_name}
```

```sh
$ exit
$ docker-compose down
$ docker-compose up -d
```
